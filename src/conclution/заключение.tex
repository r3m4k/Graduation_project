\chapter{Заключение}

В рамках производственной практики осеннего семестра были выполнены следующие задачи:

\begin{enumerate}
	\item Разработан графический интерфейс программы для ПК по взаимодействию с МК;
	\item Начата разработка алгоритма по обработке данных с датчиков, полученных от МК;
	\item Изучена система автоматизированного проектирования печатных плат KiCad;
	\item Спроектирована плата питания и подключения внешних модулей для платы МК STM32f3discovery;
	\item Установлены электронные компоненты на произведённую в ООО \enquote{Резонит} плату питания и проверена её работоспособность;
\end{enumerate} 


В весеннем семестре планируется выполнить следующие этапы работы:

\begin{itemize}
	\item Разработать корпус для платы питания и подключения внешних устройств;
	\item Завершить алгоритм обработки данных;
	\item Провести испытания разработанной системы.
\end{itemize}
