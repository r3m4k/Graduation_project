\section{Разработка ПО для постобработки данных}

В рамках проекта было начато создание специализированного программного обеспечения для углублённого анализа данных, полученных от измерительной системы. Целью данного ПО является автоматизация обработки записанных массивов данных, вычисление интегральных характеристик геометрии пути, таких как продольный и поперечный профиль и кривизна, визуализация результатов и формирование структурированных отчётов.

На текущем этапе разработки выполнена значительная часть подготовительной работы. Были спроектированы и реализованы следующие ключевые программные компоненты:


\begin{itemize}
	\item Модуль чтения и парсинга данных: Разработан класс для загрузки и интерпретации бинарных файлов логов, формируемых микроконтроллером;
	\item Модуль базовой визуализации: Создан набор классов для построения графиков временных рядов (ускорения, угловые скорости, углы ориентации) с использованием библиотек Matplotlib и NumPy.
\end{itemize}


Основная архитектура приложения построена, однако для достижения полной функциональности, соответствующей техническому заданию, требуется дальнейшая разработка, которая запланирована на весенний семестр.
