#### Стоимость блока с инерциальными датчиками и управляющим контроллером

| Наименование                     | Количество (шт) | Общая стоимость (руб) |
| -------------------------------- | --------------- | --------------------- |
| Микроконтроллер STM32f3discovery | 1               | 7000                  |
| DC-DC преобразователь            | 1               | 500                   |

```dataviewjs
// Получаем текущий файл
const page = dv.current()

// Ищем таблицы в содержимом файла
const content = await dv.io.load(page.file.path)
const tableRegex = /\|.*\|\n\|.*\|\n((?:\|.*\|\n)*)/g
let match
let total = 0

while ((match = tableRegex.exec(content)) !== null) {
    const rows = match[1].trim().split('\n')
    rows.forEach(row => {
        const cells = row.split('|').filter(cell => cell.trim() !== '')
        if (cells.length >= 2) {
            const cost = parseInt(cells[2].trim().replace(/\D/g, ''))
            if (!isNaN(cost)) {
                total += cost
            }
        }
    })
}

dv.paragraph(`**Общая сумма комплектующих: ${total} руб**`)
```
