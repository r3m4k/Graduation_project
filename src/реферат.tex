\chapter*{Реферат}


Отчет -- с., -- рис., -- источников. 

СИСТЕМА КОНТРОЛЯ ГЕОМЕТРИЧЕСКИХ ПАРАМЕТРОВ ЖЕЛЕЗНОДОРОЖНОГО ПОЛОТНА, СИСТЕМА АВТОНОМНОЙ НАВИГАЦИИ, МИКРОМЕХАНИЧЕСКИЕ ДАТЧИКИ.

Объектом исследования являются системы дефектоскопии железнодорожного полотна. 

Предмет исследования: система контроля геометрии железнодорожного пути, на базе путеизмерительной тележки.

Целью исследования является разработка инерциальной навигационной системы на основе микромеханических акселерометра и гироскопа для путеизмерительной тележки, предназначенная для контроля геометрических параметров железнодорожного полотна.

Метод проведения работы включает в себя создание программного обеспечения как для микроконтроллера STM32f3discovery, так и для персонального компьютера, разработку печатной платы для коммутации всех узлов системы и конструирование корпуса, обеспечивающий надёжность и удобство использования.

Разработанная система позволит сократить себестоимость путеизмерительной тележки, разрабатываемой в ЗАО \enquote{ПИК Прогресс}, а также снизить снаряжённую массу телеги, что является одной из основных особенностью разработки.
